# 去中心化漫画社交市场 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品背景
随着NFT技术和去中心化存储的发展，数字艺术品所有权确权变得简单可行。漫画作为重要的数字内容形态，需要一个既能保护创作者权益，又能促进社交互动和二次创作流通的平台。传统漫画平台存在创作者收益低、版权不清晰、粉丝无法参与IP共建等问题。

### 1.2 产品愿景
打造一个**创作者拥有完全所有权、粉丝可参与IP共建、跨链流通无障碍**的去中心化漫画社交生态，让漫画IP真正回归创作者和社区。

### 1.3 产品使命
- **赋能创作者**：通过智能合约确保创作者获得持续版税收入
- **连接粉丝**：让粉丝从被动观看者变为IP共建者
- **释放价值**：实现漫画资产在多个区块链间的自由流通

### 1.4 目标用户

| 用户类型 | 描述 | 核心需求 | 用户规模预估 |
|---------|------|---------|-------------|
| **主创作者** | 漫画家、插画师、编剧 | 作品确权、持续变现、IP保护 | 核心用户，初期100人 |
| **二次创作者** | 同人作者、上色师、翻译者 | 获得授权、赚取收益、粉丝积累 | 增长型，初期500人 |
| **读者/收藏家** | 漫画爱好者、NFT收藏者 | 收藏正版、社交互动、投资增值 | 基础用户，初期2000人 |
| **IP授权方** | 寻求商业授权的品牌方 | 获取IP授权、合作开发 | 商业用户，初期20家 |

### 2.2 核心功能详情

#### 2.2.1 创作者工坊模块

| 功能点 | 优先级 | 详细描述 | 验收标准 |
|--------|--------|----------|----------|
| **漫画铸造** | P0 | 支持上传多页漫画（PDF/图片），填写作品信息（名称、描述、标签），设置版税比例（一级市场0-30%，二级市场0-10%），铸造为NFT | 铸造成功后在区块链上生成唯一Token ID，IPFS存储内容不可篡改 |
| **IP家族管理** | P1 | 查看所有衍生作品列表，设置衍生品版税分成比例（默认原作可分得衍生品销售额的5%），管理授权白名单 | 创作者可随时调整分成比例，链上记录不可篡改 |
| **协作悬赏发布** | P2 | 发布翻译、上色、配音等悬赏任务，锁定赏金（USDC/ETH），设置任务要求和验收标准，自动释放赏金给验收通过的贡献者 | 悬赏任务状态可追踪，验收后资金自动划转 |

**铸造流程图**：
开始 → 连接钱包 → 上传漫画文件(IPFS) → 填写元数据 →
设置版税比例 → 预估Gas费 → 确认铸造 → 交易上链 →
生成NFT → 同步到数据库 → 展示在创作者中心

#### 2.2.2 市场交易模块

| 功能点 | 优先级 | 详细描述 | 验收标准 |
|--------|--------|----------|----------|
| **一级发行** | P0 | 创作者首发销售，支持固定价格模式，可设置发售数量、发售时间、每人限购数量 | 发售开始后用户可购买，售完自动关闭，资金自动结算给创作者 |
| **二级市场** | P0 | 用户间点对点交易，支持固定价格挂单，支持出价功能，交易时自动计算版税并分给原作和衍生创作者 | 每次交易自动拆分版税，链上记录清晰可查 |
| **批量交易** | P1 | 支持多本漫画打包购买，支持批量挂单/取消，支持批量转移 | 单次交易最多支持10个NFT，Gas费优化降低30% |
| **拍卖模式** | P2 | 支持英式拍卖（价高者得），设置起拍价、保留价、拍卖时长 | 拍卖结束时自动成交，资金自动划转 |

**交易流程图**：
卖家：连接钱包 → 选择NFT → 设置价格 → 授权市场合约 → 挂单上链 → 等待成交
买家：浏览市场 → 选择NFT → 确认价格 → 支付购买 → 交易确认 → NFT到账
合约：检查余额 → 划转NFT → 拆分版税 → 支付卖家 → 退还多余款项

#### 2.2.3 社交互动模块

| 功能点 | 优先级 | 详细描述 | 验收标准 |
|--------|--------|----------|----------|
| **持有者社区** | P1 | 仅NFT持有者可加入的专属Discord/Telegram社群，通过钱包签名验证身份，持有不同NFT享有不同身份组 | 验证通过后自动分配角色，非持有者无法访问 |
| **二次创作授权** | P2 | 粉丝申请二创，填写申请表单（创作类型、预计发布时间），创作者审核通过后合约自动生成授权凭证，记录版税分成比例 | 授权凭证上链，后续二创作品自动关联原作 |
| **创作者粉丝经济** | P2 | 粉丝可订阅创作者（月费模式），获得独家内容和空投，持有者可根据持有数量获得治理投票权 | 订阅费用自动分账，投票权重与持有量挂钩 |
| **点赞评论** | P2 | 对漫画作品进行点赞、评论，评论上链（可选），点赞数链下统计 | 链下数据实时更新，评论上链需支付Gas |

#### 2.2.4 跨链桥接模块

| 功能点 | 优先级 | 详细描述 | 验收标准 |
|--------|--------|----------|----------|
| **资产跨链转移** | P1 | 支持ABR链与以太坊/Solana/BSC等主流链互转，用户选择源链和目标链，锁定源链资产，目标链铸造封装资产 | 跨链时间<10分钟，成功率>99.5%，资产锁定安全 |
| **跨链版税追踪** | P2 | 资产跨链后原版税配置保持不变，跨链交易时版税信息同步传递，目标链市场合约识别版税信息 | 跨链后市场交易仍能正确分配版税给原创作者 |
| **跨链状态监控** | P2 | 提供跨链交易状态查询页面，显示当前进度（已锁定、中继中、已完成），失败交易支持手动重试/回滚 | 状态更新及时，失败处理机制完善 |

**跨链流程图**：
用户发起 → 源链合约锁定NFT → 跨链中继监听事件 →
跨链验证 → 目标链合约铸造封装NFT → 完成通知
